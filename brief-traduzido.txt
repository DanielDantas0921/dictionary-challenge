Apresentação

Seu desafio é construir este aplicativo da web de dicionário usando a API de dicionário e fazer com que pareça o mais próximo possível do design.

Você pode usar qualquer ferramenta que desejar para ajudá-lo a completar o desafio. Então, se você tem algo que gostaria de praticar, sinta-se à vontade para tentar.

Seus usuários devem ser capazes de:

 Pesquise palavras usando o campo de entrada
 Veja a resposta da API do Dicionário Gratuito para a palavra pesquisada
 Veja uma mensagem de validação de formulário ao tentar enviar um formulário em branco
 Reproduza o arquivo de áudio de uma palavra quando estiver disponível
 Alternar entre fontes serif, sans serif e monoespaçadas
 Alternar entre temas claros e escuros
 Veja o layout ideal para a interface dependendo do tamanho da tela do dispositivo
 Veja os estados de foco e foco de todos os elementos interativos na página
 Bônus: escolha o esquema de cores correto para eles com base nas preferências do computador. Dica: a pesquisa prefere esquema de cores em CSS.

Baixe o projeto e acesse o arquivo README.md. Isso fornecerá mais detalhes sobre o projeto e ajudará você a se preparar.

Quer algum apoio no desafio? Junte-se à nossa comunidade e tire dúvidas no canal de ajuda.
